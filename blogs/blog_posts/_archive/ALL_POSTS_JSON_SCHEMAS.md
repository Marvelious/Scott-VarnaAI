### Common Mistakes to Avoid:
- ❌ Forgetting the `<script type="application/ld+json">` wrapper
- ❌ Using markdown code blocks (triple backticks) instead of HTML script tags
- ❌ Leaving placeholder URLs instead of actual image/page URLs
- ❌ Not validating JSON syntax before publishing
- ❌ Placing schemas in wrong post (mixing up blog posts)
