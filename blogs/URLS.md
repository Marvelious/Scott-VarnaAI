## Link Format Rules

**Internal Links** (7 per post):
```html
<a href="/secure-ai-services-gdpr/" target="_blank">our AI services</a>
```

**External Authority Links** (3 per post - DOFOLLOW):
```html
<a href="https://www.gartner.com" target="_blank" rel="noopener">Gartner Research</a>
```

**NEVER use nofollow on authority citations** - blocks SEO benefit!
