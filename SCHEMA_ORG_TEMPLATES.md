## Implementation Instructions

### WordPress Implementation (All Sites):

1. **Install Rank Math SEO Plugin** (if not already installed)
2. **Navigate to:** Rank Math → General Settings → Edit Robots.txt
3. **Add Schema to Footer:**
   - Go to: Appearance → Theme Editor → footer.php
   - Insert before closing `</body>` tag:

```php
<script type="application/ld+json">
<?php
// Paste appropriate schema JSON here
?>
</script>
```

### Alternative: Use Rank Math Schema Builder
1. Navigate to each page
2. Click "Schema" tab in Rank Math metabox
3. Add "Organization" schema type
4. Fill in fields from templates above
