## Implementation Instructions (Rank Math)

### To REMOVE a Focus Keyword:
1. Edit the page in WordPress
2. Open Rank Math SEO panel (sidebar)
3. Delete the text in "Focus Keyword" field
4. Save/Update the page

### To CHANGE a Focus Keyword:
1. Edit the page in WordPress
2. Open Rank Math SEO panel (sidebar)
3. Replace existing keyword with new transactional keyword
4. Update SEO Title to include new keyword
5. Update Meta Description with call-to-action
6. Save/Update the page

### SEO Title Templates:
```
German: [Keyword] | [Benefit] | [Brand] 2025
English: [Keyword] - [Benefit] | [Brand]

Examples:
- KI Projektmanagement Software kaufen | Kostenlos testen | AI Projektmanager
- AI Marketing Services Bulgaria Quote | Free Consultation | AI Marketing BG
```

### Meta Description Templates:
```
German: [Keyword] jetzt entdecken. [Benefit]. [CTA] ✓ [Trust signal]
English: Discover [Keyword]. [Benefit]. [CTA] ✓ [Trust signal]

Examples:
- DSGVO Projektmanagement kostenlos testen. 100% konform. Jetzt Demo anfragen ✓ Made in Germany
- AI Marketing Demo Bulgaria. GDPR compliant. Request free quote ✓ EU-based
```
