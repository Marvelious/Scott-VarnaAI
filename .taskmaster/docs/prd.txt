## Technical Requirements

### Infrastructure Setup
- Docker and Docker Compose installation
- Traefik reverse proxy for SSL/TLS termination
- Nginx Alpine containers for static hosting
- Let's Encrypt SSL certificates (automatic via Traefik)

### Resource Allocation
- Total RAM budget: 8GB (CX32 specification)
- Per-app allocation: 64-128MB (static sites)
- Traefik overhead: 150MB
- System overhead: 1GB
- Available headroom: >6GB for future expansion

### DNS Configuration
- Configure A records in All-Inkl DNS management
- Subdomains pointing to Hetzner VPS IP
- SSL certificates via Traefik/Let's Encrypt

## Deployment Phases

### Phase 1: Infrastructure Preparation
- Order and configure Hetzner CX32 VPS
- Install Docker, Docker Compose, UFW firewall
- Deploy Traefik reverse proxy
- Test basic connectivity

### Phase 2: Ready Apps Deployment
- Deploy FwChange (landing page exists)
- Deploy RetirementAI (landing page exists)
- Configure DNS for both apps
- Verify SSL certificates

### Phase 3: Build Required Assets
- Build C3 Compliance React frontend to static
- Create VarnaAI Intelligence landing page
- Assess SEO Agent for static conversion

### Phase 4: Complete Deployment
- Deploy remaining 3 apps
- Configure all DNS entries
- Performance testing
- Update portfolio links on varnaai.com

## Success Criteria
- All 5 apps accessible via HTTPS with valid SSL
- Page load time < 2 seconds
- Lighthouse score > 90/100
- Mobile responsive on all sites
- Zero backend dependencies (pure static)
- Monthly cost under €25

## Constraints
- No databases required (static only)
- No authentication systems
- No signup functionality
- No Ollama or AI processing on server
- Minimal resource usage (optimize for cost)

## Risk Mitigation
- Create local backups before deployment
- Test locally with Docker before VPS deployment
- Implement monitoring (UptimeRobot free tier)
- Document rollback procedures
- Keep deployment scripts for reproducibility

## Timeline

- Day 4-5: Create missing assets
- Day 6: Deploy remaining apps
- Day 7: Testing and optimization
