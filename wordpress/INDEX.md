## WordPress Workflow

### Page Creation Process

1. **Create Blank Page**: Seiten > Neu hinzufugen
2. **Wait for Design**: User adds Kadence blocks
3. **Plan SEO**: Keywords, links, structure
4. **Write Content**: Fill all text areas
5. **Verify SEO**: Check Rank Math score (target 70-80+)
6. **Notify User**: Draft ready for review

### SEO Requirements

- **Word Count**: 600-650 words minimum
- **Focus Keyword**: 7-8 mentions (1% density)
- **External Links**: 1-2 DoFollow to authorities (BSI, BfDI)
- **Internal Links**: 2-3 to other site pages
- **Paragraph Start**: Focus keyword in paragraph 2/3

### Schema Rules

**CRITICAL**: Never create Schema.org markup without user approval.

Schema templates are in `schema-templates.md` but require explicit user request.
